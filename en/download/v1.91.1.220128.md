# Kangaroo v1.91.1 (Dev)
This is a technical preview version, please don't use it in the production environment.

## New features & bug-fixes
1. Refactor workbench and workspace
2. Refactor import/export
3. Refactor intellisense
4. Refactor toolbars
5. Refactor app expire notification
6. Improve titlebar skin support
7. Improve dialog style
8. Split setting into app setting and server setting
9. Move intellisense data to user document folder
10. Update Chinese translation
11. Fixed: can't open url with default app
12. Fixed: insert sql error(user_name -> user__name)
13. Fixed: model view crashed
14. Fixed: DSV formats has a extra column
15. Fixed: scalable image size issue
16. Fixed: warning messages

## Crash issues existed
1. PostgreSQL: OID type not yet supported(libgda), it will crash.
