# 袋鼠数据库工具 v1.97.1 (Dev)
这是一个技术预览版，供用户测试和体验新特性，因功能特性已趋近达成目标，整体质量日趋成熟，故推荐所有用户使用；生产环境使用仍需谨慎评估。

## 新特性或修复的缺陷列表
- 更新 Windows 打包脚本，修复依赖库缺失问题
- 更新 ArchLinux 打包脚本，修复依赖库缺失问题
- 升级关闭确认对话框
- 重构网格组件（GtkColumnView）
- Windows 安装脚本增加 ODBC 驱动支持
- 改进 Windows 安装包本地化支持
- 改进对象浏览视图的排序支持
- 更新中文语言支持
