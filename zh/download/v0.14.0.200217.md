# Kangaroo v0.14.0.200217

## 新特性或修复的缺陷列表
1. SQLite: 支持创建新数据库
2. 重构数据库树向下文菜单
3. 集成OpenSSL用于支持保护用户密码
4. 右侧面板图标添加手指指针，用于提示可点击
5. 修复：第一次启动时的窗口大小
