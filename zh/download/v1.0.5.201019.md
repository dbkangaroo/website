# 袋鼠数据库工具 v1.0.5 (Beta)

## 新特性或修复的缺陷列表
1. 实现单元格值上下文菜单
2. 改进数据库空值显示和编辑支持
3. 改进单元格数据文本视图，支持编码检测和转换
4. 改善中国大陆用户可访问性：显示版本更新列表
5. 提升 MySQL/MariaDB 连接稳定性
6. 修复: 切换单元格数据查看视图错误
7. 修复: 复制数据列长短名错误
