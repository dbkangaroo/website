# 袋鼠数据库工具 v0.21.1 (Beta)

## 新特性或修复的缺陷列表
1. 重构整个DataGrid，为了更好的用户体验
2. DataGrid 支持三种编辑模式: LIVE / CACHE / TRANSACTION
3. DataGrid 支持复制粘贴多行操作
4. 当粘贴多行时，自动更新编辑模式为 CACHE 模式
5. 修复: 查询构建器 where 子句表达式错误
