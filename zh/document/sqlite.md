# SQLite 特色功能

## 快速建立数据库连接
把已经存在的 SQLite 数据库文件拖放到连接首页，即可一步到位创建数据库连接

## 如何创建 SQLite 数据库
袋鼠 支持两种方式创建 SQLite 数据库文件：
1. 在文件目录中新建一个空文件并将扩展名改为 SQLite 文件扩展名(.db/.sqlite)，然后拖放到连接首页即可创建数据库连接；

2. 点击新建数据库连接，在弹出的对话框中选择 SQLite 数据库类型，然后点击左边的图标新建数据库，即可完成创建数据库文件；
